# Digest: Amazon Quick / Quick Sight の BI・ダッシュボード・チャット UX 漸進改善

## 共通テーマ

QuickSight からリブランドされた **Quick Sight / Quick Suite** の、ダッシュボード表現・データ可視化・チャット体験まわりの**incremental な機能追加**群（2025-10〜2026-05）。単発では小粒だが、合わせて見ると「BI の表現力強化」「チャット/エージェントのパーソナライズ・自動化」の 2 軸で着実に拡充が続いている。

## 各 item の要点

- **Custom Sort for Filter Controls**（2026-04-29）— フィルターコントロールのカスタムソート。
- **Sparklines in Tables**（2026-04-03）— テーブル内インラインのトレンド可視化（スパークライン）。
- **Sheet Tooltips**（2026-04-15）— シートのリッチな文脈ツールチップ。
- **Dashboards from natural language prompts**（2026-05-04）— 自然言語プロンプトからダッシュボード/分析を生成。
- **Dashboard customization: tables & pivot tables**（2025-11-21）/ **expands tables & pivot**（2026-01-20）— テーブル・ピボットテーブルの装飾拡張。
- **Dashboard Theme Customization 拡張**（2025-11-20）/ **Font customization 拡張**（2025-10-13）— テーマ・フォントのカスタマイズ強化。
- **Expanded SPICE datasets**（2026-01-20）— SPICE のサイズ拡大・取り込み高速化・データ型拡充。
- **User Preferences for chat personalization**（2026-03-09）— チャットのパーソナライズ設定。
- **Scheduling for Quick Flows**（2025-11-25）— Quick Flows のスケジュール実行。
- **Memory for chat agents**（2025-12-16）— チャットエージェントのメモリ対応。
- **Easy Resolution of Ambiguous Map Locations**（2026-02-03）— 地図上の曖昧な地点の解決。

## 推奨アクション

- BI 既存ユーザーは、テーブル/ピボット/テーマ/フォント系の改善でダッシュボードの作り直し余地を確認。
- チャット/エージェント運用者は User Preferences・Memory・Scheduling でパーソナライズ＆定期実行を検討。
