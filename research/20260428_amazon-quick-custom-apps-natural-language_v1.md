# Build custom applications using natural language in Amazon Quick (Preview)

## 要約

Amazon Quick で、自然言語の指示だけで数分でカスタム Web アプリケーションを作成できる新機能がプレビュー提供。ノーコードでライブデータに接続し、複雑なワークフローや AI 機能を埋め込んだ対話型アプリをワンクリック共有できる。

## 詳細

- **何が新しい**: 必要なものを言葉で説明するだけで、完全に対話的なアプリが生成される（コーディング不要）。ライブデータソースに接続、複雑なワークフロー実装、AI 機能の埋め込み、ワンクリックでの公開・共有。
- **ユースケース**: 営業リーダーが CRM 等からリアルタイムにデータを取得するパイプラインレビュー用アプリを作成、財務マネージャーが QuickBooks/Excel/社内システムを集約して月次クローズを簡素化、など。
- **アクセス**: 無料・AWS アカウント／クレジットカード不要でサインアップ可。5 分未満のガイド付きオンボーディング、ロール別ワークフロー（sales/marketing/finance/HR ほか）。
- **位置づけ**: BI・自動化に続き「アプリ構築」も自然言語化。市民開発（citizen development）領域への踏み込み。
