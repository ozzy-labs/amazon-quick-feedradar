# Amazon Quick now available as a desktop application for macOS and Windows (Preview)

## 要約

Amazon Quick が macOS / Windows 向けのネイティブデスクトップアプリ（プレビュー）として提供開始。ブラウザを超えてローカルファイル・OS 通知・デスクトップ操作にアクセスし、web/desktop でメモリ・知識グラフ・エージェントを共有する。

## 詳細

- **何が新しい**: ブラウザ版を超え、PC の機能を活用するネイティブアプリ。ローカルファイルへの直接アクセス、OS レベルのプロアクティブ通知、ネイティブなデスクトップ制御。
- **主な機能**: ファイルをアップロードせず読み取り・操作、アクションアイテム／カレンダー競合／メッセージの通知、ブラウザ・デスクトップアプリのタスク自動化。人・プロジェクト・関係を学習する personal knowledge graph を構築し文脈を蓄積。
- **ビルダー向け**: ローカルの Model Context Protocol (MCP) 接続で coding agent と連携可能。メモリ・知識グラフ・エージェントは web/desktop 間で共有され、文脈がサーフェスを跨いで持ち運ばれる。
- **提供**: 全 Quick サブスクライバ向けプレビュー、macOS/Windows、US East (N. Virginia) のみ。
